the shape of movies

The "shape" of Matrix


The Shape of Movies is a website for a better visualization of the colors of a film. I handled the development of script to get the color data and the website to display it in a fun way. The design of the website is inspired by the plainness of scientific papers.

Demo look for a search request on the website
